# ClawHub 列表文案

首批公开发布时，使用下面的显示名称和描述。

## `dream-memory`

- 名称：`CC Dream Memory`
- 简短描述：`把最近会话、日志和主题 memory 合并为一个简短稳定的 memory 索引，同时避免保存过期的代码事实。`
- 标签：`latest,memory,agent`

## `memory-extractor`

- 名称：`CC Memory Extractor`
- 简短描述：`从最近对话中提取稳定的用户偏好、反馈、项目和参考记忆，同时避免把脆弱的代码状态细节写进去。`
- 标签：`latest,memory,agent`

## `structured-context-compressor`

- 名称：`CC Context Compressor`
- 简短描述：`把长 coding session 压缩为九段式 continuation summary，并保留请求、文件、错误、用户消息和下一步。`
- 标签：`latest,context,agent`

## `verification-gate`

- 名称：`CC Verification Gate`
- 简短描述：`在实现后执行只读 verification pass，将 verified、unverified 和 failed 工作明确区分。`
- 标签：`latest,verification,agent`

## `swarm-coordinator`

- 名称：`CC Swarm Coordinator`
- 简短描述：`通过 research、synthesis、implementation 和 verification 的边界化 worker 分工来协调多个 agent。`
- 标签：`latest,multi-agent,agent`

## `kairos-lite`

- 名称：`CC Kairos Lite`
- 简短描述：`添加轻量 proactive job，并提供 schedule、sleep、brief 和 expiry 规则，但不做完整 daemon 平台。`
- 标签：`latest,automation,agent`

# ClawHub Listing Copy

Use the following display names and descriptions for the first public batch.

## `dream-memory`

- Name: `CC Dream Memory`
- Short description: `Consolidate recent sessions, logs, and topic memories into a short durable memory index without storing stale code facts.`
- Tags: `latest,memory,agent`

## `memory-extractor`

- Name: `CC Memory Extractor`
- Short description: `Extract durable user, feedback, project, and reference memories from recent turns while avoiding fragile code-state details.`
- Tags: `latest,memory,agent`

## `structured-context-compressor`

- Name: `CC Context Compressor`
- Short description: `Compress a long coding session into a nine-part continuation summary that preserves request, files, errors, user messages, and the next step.`
- Tags: `latest,context,agent`

## `verification-gate`

- Name: `CC Verification Gate`
- Short description: `Run a read-only verification pass after implementation so verified, unverified, and failed work are clearly separated.`
- Tags: `latest,verification,agent`

## `swarm-coordinator`

- Name: `CC Swarm Coordinator`
- Short description: `Coordinate multiple agents through research, synthesis, implementation, and verification with bounded worker ownership.`
- Tags: `latest,multi-agent,agent`

## `kairos-lite`

- Name: `CC Kairos Lite`
- Short description: `Add lightweight proactive jobs with schedule, sleep, brief, and expiry rules without committing to a full daemon platform.`
- Tags: `latest,automation,agent`
